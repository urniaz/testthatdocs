NOTE Generates global and per-section test lists.
TIP Function is idempotent and safe to rerun.

Numbering and Templates

  • {g} – global index
  • {s} – section index
  • {i} – local index
  • {l} – line index

Templates:

  • simple → {g}
  • advanced → {g}.{s}.{i}
  • full → {g}.{s}.{i}.{l}
